Preguntas

¿Por qué es una buena práctica separar el controlador de la vista?

Porque permite crear software que sea más robusto con un mejor ciclo de vida; se potencia la facilidad de mantenimiento, reutilización de código y la separación de conceptos. Le da estructura y orden al momento de mantener y trabajar la página.

Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?

$_FILES: Un array asociativo que permite cargar items al script por medio del método POST.

$_REQUEST: Un array asociativo que solicita e incluye los contenidos de $_GET, $_POST y $_COOKIE.

$_SERVER: Un array asociativo que incluye la información de headers, paths y locaciones de scripts.

Explora las funciones de php, y describe 2 que no hayas visto en otro lenguaje y que llamen tu atención.

memory_get_usage(): sirve para comprobar el uso actual de memoria. No elimina nada, solo muestra la información, para poder administrar mejor de manera manual el cerrar procesos.

uniqid(): permite generar un valor único que jamás se repite; sirve para cuando se necesita dar acceso temporal con seguridad añadida y que no se dupliquen conexiones.

¿Qué es XSS y cómo se puede prevenir?

Es inyectar código sobre algún campo de entrada de datos que una página web ofrezca para ejecutarlo, generando resultados inesperados para el diseñador de software. Para prevenirlo, debemos filtrar las entradas de usuario para evitar que se introduzcan secuencias de caracteres escritas en HTML. También se deben implantar y configurar cortafuegos de aplicación web (WAF), que filtran los ataques que escapen la primera línea de defensa.
