Bases de datos

¿Qué es entonces una base de datos - DB?

Es un almacén que nos permite guardar grandes cantidades de información de forma organizada para hallarla y hacer uso de ella con facilidad. Es una colección de archivos interrelacionados de manera que sea lo menos redundante, repetitivos, y que se pueda hacer uso general de ellos.

¿En qué casos es conveniente usar bases de datos?

Siempre que la información que se maneja presenta varios archivos interrelacionados. Se dice que hoy en día, cualquier organización que se precie debe contar con una. Es importante para que la empresa o institución mantenga sus propias relaciones internas, y ofrecer un alto rango de soluciones a problemas de almacenar datos.

¿Qué es un sistema de gestión de base de datos - DBMS?

Son un conjunto de programas que realizan tareas de manera interrelacionada para facilitar la construcción y manipulación de bases de datos, adoptando la forma de interdaz entre ellas, las aplicaciones y los usuarios.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  1. Data Store Management: la función se encarga del almacenamiento de los datos, validación de datos, cómo se almacena y se manipulan los datos.
  2. Data Transformation and Presentation: Se usa para transformar los datos entrantes en las estructuras de datos requeridas para asegurar el funcionamiento eficiente.
  3. Security Management: especifica qué usuarios tienen permitido acceder a la base de datos. Se controla con un nombre de usuario y contraseña.
  4. Multiuser Access Control: Permite a varios usuarios acceder a la base de datos simultáneamente sin afectar su integridad
  5. Backup and Recovery Management: permite restablecer la base de datos a una versión anterior en caso de que alguna amenaza externa se presente.
  6. Data Integrity Management: reduce la redundancia de datos.
  7. Transaction Management: garantiza que las actualizaciones de una transacción se realizarán del todo o no.
  8. Database Communication Interfaces: permite a los usuarios acceder por diferentes web browsers, por ejemplo.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

Si hubiera ido más allá de mi proyecto de Diseño Interactivo que sólo la fase de análisis y diseño, probablemente hubiera sido conveniente utilizar una base de datos. Porque nos hubiera ayudado a mantener mayor orden en cuanto a aquello lo que los usuarios requerían, entender mejor su interacción con el sitio, y habríamos mantenido más organizados los datos que recopilábamos porque simplemente llenábamos hojas de cálculo.
