¿Qué diferencias y semejanzas hay entre Java y JavaScript?

JavaScript es un lenguaje de programación desarrollador por Netscape, Inc. que permite "instruir" a la computadora apenas recibe input del usuario. Primero llamado "LiveScript, se creó alrededor de 1990 y era indispensable para navegar por Internet.Java fue lanzado alrededor de 1995 y es u lenguaje de programación desarrollado por Sun Microsystems.

El código de Java debe compilarse, mientras que el código de JavaScript es puro texto. Cada lenguaje requiere plug-ings diferentes. JavaScript corre solo en un navegador, mientras que Java crea aplicaciones que corren en máquinas virtuales y browsers. Java y JavaScript son ampliamente usados en los dispositivos inteligentes de hoy.[5]

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  1. getDate() : Regresa el día del mes.
  2. getFullYear() : Regresa el año
  3. getHours(): Regresa la hora (del 0 a 23)
  4. getTime(): Regresa el número de milisegundos desde medianoche Enero 1 1970, y una fecha especificada.
  5. getTimezoneOffset(): Regresa la diferencia de tiempo entre tiempo UTC y el tiempo local, en minutos.[4]

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  1. toString(): convierte un array a string separados por comas.
  2. pop(): saca el último elemento de un array.
  3. push(): añade un nuevo elemento al array al final.
  4. splice(): inserte y remueve elementos, permitiendo especificar la posición de los nuevos elementos que serán insertados.
  5. concat(): crea un nuevo array juntando dos arrays existentes.[3]

¿Cómo se declara una variable con alcance local dentro de una función?

Se debe de anteponer la palabra reservada var, seguido del nombre por asignar a la variable. Esa variable tendrá solo alcance local, aunque también es posible delimitarlo con la palabra reservada let.[1]

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Implica que debes tener en claro que es visible para todas las funciones y que de preferencia deben ser valores que sean constantes para todas ellas. Si no se toma esto en cuenta, podría usarse una variable que no se le reasigna un valor, utilizar el valor que obtuvo en una función anterior y generar errores inesperados en el código.

¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?

El método match(). Podría utilizarse para buscar un dato específico en una lista, si el usuario quisiera visualizarlo directamente en vez de buscarlo; podría usarse algo similar al Control + F o Command + F que utilizamos para buscar una palabra en específico. [2]
