¿Qué alternativas a jQuery existen?

No es necesario usar jQuery para poder implementar AJAX. De hecho, muchas de las funcionalidades generalmente requeridas pueden concretarse con las librerías modernas de 'Vanilla' JavaScript que permiten hacer llamadas asíncronas. Sin embargo, algunas alternativas son: Cash (una alternativa ligera a jQuery para browsers modernos, con una sintaxis similar a jQuery y que ocupa apenas 32 KB de espacio, sin comprimirse), Zepto (una librería de JavaScript minimalista con un API ampliamente compatible con jQuery) y Syncfusion Essential JS 2 (controles UI de JavaScript escritos en TypeScript).

¿Qué es una promesa en js?

Una promesa es un objeto que puede o no devolver algún valor en la línea de tiempo presente y fututo. Se ejecuta código asíncrono y se obtiene la promesa de que se obtendrá una respuesta, que puede ser en ese instante o en un futuro. Gracias a este comportamiento, es posible implementar funciones asíncronas que den respuesta sin interrumpir el proceso de la página principal.

¿Cómo funcionan async y await?

async es la palabra reservada que se escribe antes de una función para convertirla en una función asíncrona. Una función asíncrona es una función que sabe cómo esperar la posibilidad de la palabra reservada await para invocar código asíncrono. Las funciones asíncronas regresan promesas.

await es la palabra reservada que solo funciona dentro de funciones asíncronas. Se pueden poner al frente de cualquier función asíncrona basada en promesas para pausar el código en esa línea hasta que la promesa se cumpla, regresando entonces el valor resultante. Mientras tanto, el resto del código que pueda estar esperando a ser ejecutado, lo hace.
